# AI Wallpaper Hub

Stunning AI-generated mobile wallpapers for Android. Built with Next.js 14, TailwindCSS, and Google Apps Script backend.

## Features
- 9:16 aspect ratio wallpapers (1080x1920)
- 10 categories: Nature, Temple, Anime, Animals, Adventure, Horror, Space, Cyberpunk, Abstract, Minimal
- AI-generated prompts via Gemini 2.0 Flash
- Images generated by Pollinations AI
- Auto-update every 1 hour (2 new wallpapers)
- Favorite wallpapers (localStorage)
- Direct download & share
- Shareable URLs: /w/neon-tiger-cyber-night

## Tech Stack
- **Frontend:** Next.js 14, TypeScript, TailwindCSS, Font Awesome
- **Backend:** Google Apps Script (doGet API), Google Sheets
- **Deployment:** Vercel (frontend), Google Apps Script Web App (backend)
- **AI:** Gemini 2.0 Flash (prompt generation), image.pollinations.ai (wallpaper)

## Setup
1. Clone this repo.
2. Set environment variable `APPS_SCRIPT_URL` in Vercel dashboard (or .env.local for local dev).
3. Deploy to Vercel.

## Backend
Google Apps Script code located in `/apps-script/Code.gs`. Deploy as Web App, then use its URL as `APPS_SCRIPT_URL`.
